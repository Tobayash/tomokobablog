同じ形のDFを持つlistを１つのDFに

05 May 2020

同じ形のデータフレームを持つリストを、リスト番号を持ったまま１つのデータフレームにしたい。 ’qdap’パッケージのlist_df2dfが使えそうなのでメモ。

詳しくは以下のリンクより https://www.rdocumentation.org/packages/qdapTools/versions/1.3.5/topics/list2df

library(qdapTools)

L <- list()
for (i in 1:5) {
  Df <- data.frame(ID = i,
                   V1 = 1:3)
  L[[i]] <- Df
}
